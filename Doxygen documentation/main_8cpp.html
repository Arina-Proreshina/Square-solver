Подробное описание

Программа для решения квадратного уравнения.

Эта программа позволяет пользователю ввести коэффициенты квадратного уравнения и вычислить его корни. Она включает в себя функции для получения ввода, решения квадратного уравнения и печати результатов.

Программа предоставляет следующие функции:

  • is_zero для проверки, близко ли значение к нулю в пределах EPSILON.
  • get_number_input для получения числового ввода от пользователя.
  • get_coefficients для ввода коэффициентов квадратного уравнения.
  • solve_linear_equation для вычисления корней линейного уравнения.
  • solve_square_equation для вычисления корней квадратного уравнения.
  • print_solution для печати результатов.
  • main для выполнения основной логики программы.
Заметки
Внимание: Функция get_number_input проверяет ввод чисел и ожидает, что ввод будет числовым значением.

Перечисление для представления количества корней квадратного уравнения.

Это перечисление используется для обозначения количества вещественных корней квадратного уравнения. Оно определяет возможные значения количества корней уравнения в зависимости от дискриминанта и коэффициентов уравнения.

Элементы перечислений

Уравнение не имеет вещественных корней.

Значение 0: Уравнение не имеет вещественных корней.

OneRoot 

Уравнение имеет один вещественный корень.

Значение 1: Уравнение имеет один вещественный корень.

TwoRoots 

Уравнение имеет два вещественных корня.

Значение 2: Уравнение имеет два вещественных корня.

InfRoots 

Уравнение имеет бесконечно много корней.

Значение 3: Уравнение имеет бесконечно много корней (применимо для случая, когда a = 0 и b = 0).

◆ solve_linear_equation()

SquareEquationResult solve_linear_equation ( SquareEquationCoefficient coeffts)

Решает линейное уравнение вида bx + c = 0.

Эта функция вычисляет корень линейного уравнения на основе коэффициентов b и c. Она обрабатывает специальные случаи, когда коэффициенты b и/или c равны нулю.

Аргументы
[in]coefftsСтруктура, содержащая коэффициенты линейного уравнения.
  • coeffts.b Коэффициент при x.
  • coeffts.c Свободный член.
Возвращает
SquareEquationResult Структура, содержащая результат решения уравнения.
  • result.x1 Значение корня уравнения (действительное число).
  • result.result_type Тип результата:
    • InfRoots Если уравнение имеет бесконечно много решений (когда b = 0 и c = 0).
    • NoRoots Если уравнение не имеет решений (когда b = 0 и c != 0).
    • OneRoot Если уравнение имеет один корень (когда b != 0).

◆ solve_square_equation()

SquareEquationResult solve_square_equation ( SquareEquationCoefficient coeffts)

Решает квадратное уравнение вида ax^2 + bx + c = 0.

Эта функция вычисляет корни квадратного уравнения на основе коэффициентов a, b и c. Она также учитывает случай, когда коэффициент a равен нулю, и переходит к решению линейного уравнения вида bx + c = 0.

Аргументы
[in]coefftsСтруктура, содержащая коэффициенты квадратного уравнения.
  • coeffts.a Коэффициент при x^2.
  • coeffts.b Коэффициент при x.
  • coeffts.c Свободный член.
Возвращает
SquareEquationResult Структура, содержащая результат решения уравнения.
  • result.x1 Первый корень уравнения (действительное число).
  • result.x2 Второй корень уравнения (действительное число, если существует).
  • result.result_type Тип результата:
    • InfRoots Если уравнение имеет бесконечно много решений (когда a = 0, b = 0 и c = 0).
    • NoRoots Если уравнение не имеет решений (когда a = 0, b = 0 и c != 0).
    • OneRoot Если уравнение имеет один корень (когда a = 0, b != 0 или дискриминант равен нулю).
    • TwoRoots Если уравнение имеет два корня (когда дискриминант больше нуля).

Переменные

◆ EPSILON

const double EPSILON = 1e-6

Константа, представляющая значение (10^{-6}).

Эта константа используется в программе для представления значения, близкого к нулю. Она может быть полезна при сравнении чисел с плавающей запятой для учета погрешности вычислений и определения, когда число считается практически равным нулю.

Заметки
Значение константы установлено как 1e-6, что эквивалентно (10^{-6}). Оно может использоваться для проверки чисел на близость к нулю, чтобы избежать проблем, связанных с точностью вычислений с плавающей запятой.
