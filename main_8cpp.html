Подробное описание

Программа для решения квадратного уравнения.

Эта программа позволяет пользователю ввести коэффициенты квадратного уравнения и вычислить его корни. Она включает в себя функции для проверки ввода, решения квадратного уравнения и печати результатов.

Программа использует следующие функции:

  • my_isdigit для проверки, является ли символ цифрой.
  • solve_square_equation для вычисления корней квадратного уравнения.
  • print_solution для печати результатов.
  • is_number для проверки, является ли строка числом.
  • get_number_input для получения числового ввода от пользователя.
  • main является главной функцией программы.
